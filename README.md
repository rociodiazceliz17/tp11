## Descripción breve
Sitio sobre 'Pueblos Originarios de la Región'. Tema: Pueblos Originarios de la Región. Autor: Rocio Diaz.

## Secciones incluidas
- Barra de navegación responsiva
- Slider con imágenes (Bootstrap Carousel)
- Sección con texto e imagen
- Cards informativas
- Galería multimedia con audio y video (archivos locales en carpetas `audio/` y `video/`)
- Formulario de contacto con nombre, email y mensaje

## Estructura de archivos (carpeta `tp11-web/`)
```
tp11-web/
  index.html
  css/styles.css
  js/scripts.js
  img/ (colocar imágenes reales aquí)
  audio/ (colocar archivos .mp3 aquí)
  video/ (colocar archivos .mp4 aquí)
```

## Cómo usar / ver el sitio
1. Descargar o clonar este repositorio.
2. Abrir `tp11-web/index.html` en un navegador web (doble click o arrastrar al navegador).
3. Reemplazar las imágenes en `img/`, y los archivos multimedia en `audio/` y `video/` por archivos reales si se desea.
4. Si querés usar videos de YouTube en lugar de archivos locales, reemplazar el elemento `<video>` por un `<iframe src="https://www.youtube.com/embed/VIDEO_ID">` y cambiar `VIDEO_ID` por el id del video.

## Fuentes recomendadas para obtener imágenes y multimedia reales
- **Wikimedia Commons:** imágenes con licencia libre. Buscar términos: "Atahualpa Yupanqui", "Cerro Colorado", "Casa Museo Atahualpa", "senderismo Argentina".
- **Internet Archive:** música y grabaciones antiguas (ver licencias).
- **YouTube:** para videos (usar la opción de embeber o descargar si la licencia lo permite).

### Nota importante sobre multimedia real
No pude descargar automáticamente imágenes o videos (sin conexión a Internet desde este entorno). Por eso este proyecto incluye imágenes y archivos multimedia *placeholder*. Para cumplir exactamente con el requerimiento de "usar imágenes y videos reales", seguí las instrucciones anteriores para descargar y colocar los archivos dentro de las carpetas `img/`, `audio/` y `video/`. Si me pegás aquí los enlaces directos (URLs) de YouTube o de archivos en Wikimedia/Internet Archive, puedo editar los `index.html` para incrustarlos directamente.
